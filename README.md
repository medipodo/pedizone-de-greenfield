# pedizone.de

Greenfield SEO project for foot health information in German.

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Yarn (package manager)
- File-based routing
- Language: German (de-DE)

## Development

```bash
yarn dev
```

## Build

```bash
yarn build
yarn start
```

## Deployment

Automatic deployment via Vercel on push to `main` branch.
